# ScaleCaptureWinFormCSharp

# 概要
【 ScaleCaptureWinFormCSharp 】  
グローバルフックを使用してマウスの座標のキャプチャーを表示するアプリケーションです。  
最近傍補間法を使用して画像を画像の拡大・縮小して画像を表示します。  
デザインはフラットデザインにしています。  

開発環境：Visual Studio 2019  
言語：C#
